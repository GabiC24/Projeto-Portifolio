Sobre mim

Gabriela Cunha: Sou apaixonada por tecnologia e por aprender constantemente. Minha jornada começou no front-end, mas descobri meu verdadeiro interesse nas áreas de Inteligência Artificial e Cibersegurança, onde estou mergulhando de cabeça.

Adoro explorar novas ideias e transformar projetos simples em algo funcional e cheio de propósito. Com uma abordagem prática e criativa, acredito que cada desafio é uma oportunidade de evolução. Enquanto codifico, bugo e aprendo, busco criar soluções que inspirem e impactem positivamente o mundo.

Quando não estou estudando ou desenvolvendo, gosto de passar tempo com minha família, ler, assistir filmes e séries e, claro, me divertir com pequenos projetos que unem simplicidade e diversão.

"Codando, bugando e aprendendo" é mais do que uma frase: é meu estilo de vida.

Cursos Complementares

Formação HTML 5 Web Developer pela DIO

Essa formação tem uma carga horária de 21 horas, composta por 13 cursos e 3 desafios de projeto. Ao longo do programa, são abordados os principais conceitos para o desenvolvimento de páginas web:

  • Primeiros passos com HTML: Introdução ao universo da marcação e estruturação de páginas.
  • HTML I: Conceitos básicos: Fundamentos essenciais para criar páginas funcionais e bem estruturadas.
  • HTML II: Ampliando o conhecimento: Exploração de técnicas mais avançadas para criar layouts mais completos.

Formação CSS Web Developer pela DIO

Com duração de 43 horas, essa formação inclui 15 cursos e 5 desafios de projeto. O foco é no desenvolvimento de habilidades para estilização e layout de páginas web. Os principais tópicos abordados incluem:

  • Primeiros passos com CSS: Introdução ao uso de estilos para transformar o visual das páginas.
  • Trabalhando com Layouts no CSS: Criação de estruturas organizadas e responsivas para melhorar a experiência do usuário.
  • Refinando os estilos CSS das nossas páginas: Técnicas avançadas para aprimorar a estética e a usabilidade de projetos web.

Formação JavaScript Developer pela DIO

Essa formação, com 39 horas de conteúdo, é composta por 20 cursos, 2 desafios de projeto e 1 desafio de código. O objetivo é dominar o JavaScript, desde o básico até conceitos avançados, aplicando o conhecimento em projetos práticos. Principais módulos:

  • Aprenda a programar do zero com JavaScript: Introdução à lógica de programação e fundamentos essenciais da linguagem.
  • JavaScript para Web: Fundamentos de HTML, CSS e APIs na prática, com foco na interação entre front-end e back-end.
  • Dominando conceitos avançados em JavaScript: Exploração de técnicas complexas para criar soluções robustas e eficientes.
  • Currículo online com HTML, CSS e JavaScript: Do planejamento inicial ao deploy, aplicando os aprendizados para criar um projeto completo.
